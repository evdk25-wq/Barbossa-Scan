BARBOSSA SCAN v1.0 BY JLVDK95

PIRATE SECURITY TOOL FOR LINUX
DESCRIPTION

Barbossa Scan is an automated security utility. It allows you to update your threat databases and scan your ship (your PC) for viruses (ClamAV) and rootkits (rkhunter). It includes a dedicated menu to target specific folders such as Desktop, Documents, or Downloads.
PACKAGE CONTENTS

    scan_securite.sh: The application engine containing the scan logic.

    pirate.svg: The official application icon.

    install_barbossa.sh: The automatic installation script.

INSTALLATION (INSTRUCTIONS)

To install the tool on your system, follow these simple steps:

    Open a terminal in the folder containing the files.

    Make the installation script executable with the following command: chmod +x install_barbossa.sh

    Run the installation: ./install_barbossa.sh

    Enter your password when prompted (to install the required security tools).

Once the installation is complete, you will find "Barbossa Scan" directly in your applications menu (System or Security category).
USAGE

Upon launch, a menu will appear to choose your target:

    1: Full Home Directory

    2: Desktop

    3: Documents

    4: Downloads

    5: Music

    6: Pictures

    7: Videos

The scan may take some time depending on the size of your loot. Do not close the window before the final report!
